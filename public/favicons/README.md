# Favicon Upload Instructions

## Folder Structure
Upload your favicon image files directly into this `public/favicons/` folder.

## Required Files

Please upload the following favicon files:

1. **favicon.svg** - SVG format (scalable, modern browsers)
2. **favicon-16x16.png** - 16x16 pixels PNG (fallback)
3. **favicon-32x32.png** - 32x32 pixels PNG (standard)
4. **apple-touch-icon.png** - 180x180 pixels PNG (iOS devices)
5. **favicon.ico** - ICO format (older browsers, optional)

## After Uploading

Once you've uploaded your favicon files to this folder, the HTML will automatically reference them.

## File Locations

- SVG favicon: `public/favicons/favicon.svg`
- PNG favicons: `public/favicons/favicon-16x16.png`, `public/favicons/favicon-32x32.png`
- Apple touch icon: `public/favicons/apple-touch-icon.png`
- ICO favicon: `public/favicons/favicon.ico` (optional)

## Notes

- Make sure all files have transparent backgrounds
- The SVG file should be the primary favicon
- PNG files are fallbacks for browsers that don't support SVG
- Apple touch icon should be 180x180 pixels for best results on iOS devices
